# Contributing | 贡献指南

Thank you for helping keep this **AI-focused** archive high-signal!  
感谢你帮助维护这份 **AI 优先** 的高信噪比归档！

## Core Principles | 核心原则

1. **AI First** — Prioritize AI / LLM / Agent / Coding Agent / RAG / MCP related projects  
2. **Separate READMEs** — Maintain both `README.md` (English) and `README.zh-CN.md` (Chinese)  
3. **Separated Heat** — Always distinguish **GitHub Heat** and **X Heat**  
4. **High Signal** — Only high-quality projects with real discussion or significant stars

## Daily Update Workflow | 每日更新流程

1. Check [GitHub Trending](https://github.com/trending) — focus on AI related
2. Search X for high-engagement AI open-source recommendations
3. Update `archives/YYYY-MM.md` (add new day at the top)
4. Update **both** `README.md` (English) and `README.zh-CN.md` (Chinese) "Today's Highlights" sections
5. Commit & push (or open a PR)

## Format Guidelines | 格式规范

```markdown
## 📅 YYYY-MM-DD

### 🔥 GitHub Heat | GitHub 热度 (AI Priority)
- [owner/repo](url) — short description (⭐ X today)

### 💬 X Heat | X 热度
- Project name — why it's hot + link
```
