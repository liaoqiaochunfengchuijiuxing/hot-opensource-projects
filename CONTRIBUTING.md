# Contributing to Hot Open Source Projects

Thank you for helping keep this archive high-signal and up-to-date!

## Daily Update Workflow (Recommended)

1. Check [GitHub Trending](https://github.com/trending) (Today)
2. Search X for discussions: `"open source" OR github (recommend OR trending OR hot) since:YYYY-MM-DD`
3. Open `archives/YYYY-MM.md` for the current month
4. Add a new `## 📅 YYYY-MM-DD` section at the top (newest first)
5. Optionally update the "Today's Highlights" table in `README.md`
6. Commit & push (or open a PR)

## Format Guidelines

Keep entries concise:

```markdown
## 📅 2026-07-13

### 🔥 GitHub Trending

1. **[owner/repo](https://github.com/owner/repo)**  
   Language · ⭐ N today · description in one sentence.

### 💬 X Discussions & Recommendations

- **Project Name** — short reason + [link to post](url)
```
