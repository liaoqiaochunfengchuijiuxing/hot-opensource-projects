# Contributing to Hot Open Source Projects | 贡献指南

Thank you for helping keep this **AI-focused** archive high-signal and up-to-date!  
感谢你帮助维护这份**AI 优先**的高信噪比归档！

## Core Principles | 核心原则

1. **AI First** — 优先收录 AI / LLM / Agent / Coding Agent / RAG / MCP 相关项目  
2. **Bilingual** — 中英双语（标题、描述、表格）  
3. **Separated Heat** — 明确区分 GitHub Heat 与 X Heat  
4. **High Signal** — 只收高质量、有实际讨论或显著热度的项目

## Daily Update Workflow | 每日更新流程

1. Check [GitHub Trending](https://github.com/trending) (Today) — focus on AI related  
2. Search X for high-engagement AI/open-source recommendations  
3. Open `archives/YYYY-MM.md` for the current month  
4. Add a new `## 📅 YYYY-MM-DD` section **at the top** (newest first)  
5. Update the "Today's Highlights" tables in `README.md`  
6. Commit & push (or open a PR)

## Format Guidelines | 格式规范

Keep entries concise and bilingual:

```markdown
## 📅 2026-07-13

### 🔥 GitHub Heat | GitHub 热度 (AI 优先)

1. **[owner/repo](https://github.com/owner/repo)**  
   Language · ⭐ N today  
   English description.  
   中文简短描述。

### 💬 X Heat | X 热度

- **Project Name** — why it's hot on X + [link](url)  
  中文原因 + 链接
```
