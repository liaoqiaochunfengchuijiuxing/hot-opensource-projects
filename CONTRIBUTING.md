# 贡献指南

## 核心规则

1. 只关注 AI 方向（Agent / LLM / Coding Agent / MCP / RAG 优先）
2. 仓库全中文
3. 每天严格三部分结构：

```markdown
## YYYY-MM-DD

### 1. GitHub 热点项目
（按今日 stars 排序的 AI 项目表格）

### 2. X 上热议项目及主流观点
（项目 + 大家的真实看法）

### 3. 重点项目深入介绍
（对每个重要项目写深入解读：是什么、为什么重要、适合谁）
```

4. README 保持干净，不要多余介绍部分。

## 每日流程
1. 从 GitHub Trending 抓取 AI 相关热门项目
2. 搜索 X 上的热议观点
3. 对重点项目写深入介绍
4. 更新 README.md + 当月归档
5. 提交
