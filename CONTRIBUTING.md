# 贡献指南

## 核心规则（必须遵守）

1. **全中文**
2. **只收 AI / Agent / RAG / Agentic 落地项目**
3. 每天固定结构（严格）：

```markdown
## YYYY-MM-DD

### 1. 今日20个高质量落地项目（GitHub + X 合并去重）
（固定20个的表格，带来源）

### 2. X 上热议观点（详细）

### 3. 重点项目深入介绍（十分详细）

### 4. 基于今日20个项目的商业点子推荐
```

4. 从 GitHub Trending + X 热议各自取约20个，去重后精选固定20个。

## 自动处理机制

本仓库已开启 **自动 PR 处理**（由 Grok 每日运行）：

- 每天会自动检查所有 open Pull Request
- 如果 PR **完全符合以上规范** 且内容高质量 → **自动 Approve + Squash Merge**
- 如果不符合 → 自动留言详细修改意见，不合并

因此，请务必严格遵循格式，否则会被自动拒绝并要求修改。

## 推荐流程
1. Fork 本仓库
2. 按最新 README 结构更新当天内容
3. 提交 PR（标题建议：`Daily update: YYYY-MM-DD` 或 `贡献：补充XX项目`）
4. 等待自动审核（通常当天或次日）

感谢贡献！
