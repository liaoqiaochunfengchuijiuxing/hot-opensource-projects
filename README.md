## 📖 About

This repository maintains a living document of popular open-source projects:

- **Every day**: Capture top trending repositories from [GitHub Trending](https://github.com/trending)
- **Every day**: Capture the open-source projects that are most discussed / recommended on X
- **Monthly rolling archive**: All daily entries are organized under monthly files
- **GPL-3.0 licensed**: Free to use, share, and contribute under the GNU General Public License v3

The goal is to build a high-signal, community-maintained archive that helps developers discover great open-source work without drowning in noise.

## 🛠️ How to Contribute

1. Fork the repo
2. Add today's entries to the current monthly archive (`archives/YYYY-MM.md`)
3. Update the "Today's Highlights" section in `README.md` if you want
4. Open a Pull Request

Or simply open an Issue with a project you think should be included.

**Suggested daily format** (copy-paste friendly):

```markdown
## 📅 YYYY-MM-DD

### GitHub Trending
- [owner/repo](url) — short description (⭐ X today)

### X Discussions & Recommendations
- Project name — why it's hot + link to post
```
