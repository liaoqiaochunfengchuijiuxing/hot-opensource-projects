## 📖 About | 关于本项目

This repository maintains a living document of popular **AI-oriented** open-source projects:  
本仓库维护一份**以 AI 方向为主**的热门开源项目动态文档：

- **Every day**: Capture top AI-related trending repositories from [GitHub Trending](https://github.com/trending)  
  **每日**：抓取 [GitHub Trending](https://github.com/trending) 中 AI / Agent / LLM / Coding Agent 相关热门仓库
- **Every day**: Capture the open-source projects that are most discussed / recommended on X  
  **每日**：记录 X 上讨论与推荐最热的开源项目（尤其 AI 相关）
- **Separated heat ranking**: Clear distinction between **GitHub Heat** and **X Heat**  
  **热度分离**：明确区分 **GitHub 热度** 与 **X 热度**
- **Monthly rolling archive**: All daily entries organized under monthly files  
  **月度滚动归档**：所有每日记录按月归档
- **GPL-3.0 licensed**: Free to use, share, and contribute  
  **GPL-3.0 开源**：可自由使用、分享与贡献

The goal is to build a high-signal, AI-first archive that helps developers discover great open-source work without drowning in noise.  
目标是打造高信噪比、**AI 优先**的归档，帮助开发者高效发现优质开源项目。

## 🛠️ How to Contribute | 如何贡献

1. Fork the repo  
2. Add today's entries to the current monthly archive (`archives/YYYY-MM.md`)  
   把今日内容添加到当前月归档  
3. Update the "Today's Highlights" section in `README.md` if needed  
4. Open a Pull Request  

Or simply open an Issue with a project you think should be included.  
也可以直接开 Issue 推荐项目。

**Suggested daily format | 推荐每日格式** (copy-paste friendly):

```markdown
## 📅 YYYY-MM-DD

### 🔥 GitHub Heat | GitHub 热度 (AI 优先)
- [owner/repo](url) — short description (⭐ X today) / 简短中文描述

### 💬 X Heat | X 热度
- Project name — why it's hot on X + link / 为什么在 X 上热 + 链接
```
