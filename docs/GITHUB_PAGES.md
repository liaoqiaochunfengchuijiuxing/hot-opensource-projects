# GitHub Pages 部署说明

本仓库已针对 GitHub Pages 优化（纯 Markdown + 索引结构）。

## 一键启用步骤（你需要手动操作一次）

1. 打开仓库：https://github.com/liaoqiaochunfengchuijiuxing/hot-opensource-projects
2. 点击 **Settings** → 左侧 **Pages**
3. **Source** 选择：
   - Branch: `main`
   - Folder: `/ (root)`
4. 点击 **Save**

几分钟后即可访问：
**https://liaoqiaochunfengchuijiuxing.github.io/hot-opensource-projects/**

## 推荐设置

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)` （直接使用 README.md 作为首页）
